My name is Catherine Jin, and I am a STEM concentration student of Class of 2022 at Foxcroft School. The purpose of this website is to provide students access and references of Foxcroft's math curriculums, which is what makes this website special.

I started this project from scratch my senior year, and it currently covers Foxcroft's Algebra I(lesson summary and practice), Algebra II(lesson summary), AP Statistics(Review guide), and AMC(practice problems).

I hope this project can be developed by other concentration students, so it would cover more curriculums and practice problems to help more students!
